User Stories

Here you will find User Stories for the Storefront Project.

Jennifer Wong

  • As a booker , Jennifer would like to be able to request meeting spaces, so that she doesn't have to e-mail the supervisors.
  • As a booker, Jennifer would like to be able to see available meeting spots, so that she has access to the information.
  • As a booker, Jennifer would like to be able to see the schedule by clients, so that she can relay the information to the client.
  • As a booker, Jennifer would like to be able to see the upcoming daily schedule, so that she can prepare for the day.
  • As a booker, Jennifer would like to be able to see the upcoming schedule in a weekly form, so that she can prepare for the week.
  • As a booker, Jennifer would like to see the upcoming schedule in a monthly form, so that she can prepare for the month.
  • As a booker, Jennifer would like to be able to see information on missed booking from a client, so she can make informed decisions.
  • As a booker, Jennifer would like to be able to put clients on a waiting list for a particular meeting space, so that she keep track of the information.
  • As a booker, Jennifer would like to be kept updated on changes to the schedule, so that she can keep organized.
  • As a booker, Jennifer would like to be able to see a history of changes, so that she can keep organized.
  • As a booker, Jennifer would like to be able to undo an action, in case she makes a mistake.
  • As a booker, Jennifer would like to be able to input client information, so that she can keep track of the clients.

  • s a booker, Jennifer would like to be able to access this information on-the-go, so that she can be on top of the Storefront schedule.
  • As a booker, Jennifer would like to be able to input room information, so that she can relay the information to the client.

Juan Martinez

  • Juan would like to view the contact details of his students so that he can contact them in the event of an emergency (i.e. class is cancelled) and not let them down.
  • As a student of Juan, I would like to receive a notification by telephone or email if a class is cancelled.
  • Juan would like to view how many students are going to attend his art class so that he can bring the necessary amount of materials.
  • Juan would like to view which rooms are available on a specific date so that he can schedule his art classes accordingly.
  • Juan would like to book a room at a specific date and time for his art class so that he can conveniently book rooms without going to Storefront, as it is not always convenient for him.
  • As a student of Juan, I would like to sign up his for his art classes online so that it is much more convenient than calling in.
  • As a student of Juan, I would like to view the details of a class that I will be signing up for so that I can see whether I will be interested in the lesson.
  • Juan would like to sign up for a cooking class with the online system so that he can secure his spot in the class.
  • Juan would like to see the number of remaining spots for the cooking class so that he can see how many friends he can invite to the class.
  • Juan would like to see a calendar view of his classes so that he can see which classes he can have a better view of which classes he will be teaching, and which ones he will be attending.
  • Juan would like a tablet friendly interface so that he can access and book rooms using his iPad, as he does not always have access to his laptop.
