Technological Pivot - NEW

  • Django - Switched to a high-level Python web framework so that database tables can be created with minimal trouble. All group members are familiar with Python as opposed with PHPMyAdmin Control.
  • Windows Azure - We're using cloud server hosting to showcase our in-progress work to the product champion because it is the most economical means available to us.
  • HTML & CSS - to create a nice user interface that is simple and lightweight. As we reach the usabilit stages, we shall look into including techniques from HTML5, CSS3, jQuery, Node.JS and other such tools.

Technology Used - OLD

  • MySQL Database with PHPMyAdmin Control - PHPMyAdmin makes it easier to control and create tables for the setup
  • cPanel - a GUI Interface Control Panel tool to faciliate web design as well as provide other powerful tools such as PHPMyAdmin
  • CakePHP as a PHP Framework Built in tools for input validation and to help implement security controls.
